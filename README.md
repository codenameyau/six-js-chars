# six-chars
http://jazcash.com/a-javascript-journey-with-only-six-characters/


```javascript
// Using just these symbols: [ ] ( ) ! +

// Simple rules.
!value     // converts to boolean.
+value     // converts to a number.
value + [] // converts to a string.

// Other rules.
'hello'[0] === 'h'
+('1' + '1') === 1
```
